# POWERBI ADVANCED ANALYTICS PORTFOLIO

This repository contains a collection of professional Power BI dashboards focused on Advanced Visualization, E-commerce Analytics, and Food Delivery Sales Performance.
The project demonstrates the ability to convert complex business data into interactive, insightful, and decision-ready analytics.

==============================
# KEY FEATURES

Advanced Visualization Report with custom visuals and visual storytelling

Complex DAX measures for YoY Growth, Rolling Averages, and Segmentation

Interactive E-commerce Funnel Analysis

Robust Data Modeling using optimized star schemas

Detailed Food Delivery Sales and Profitability Analytics

User-centric design for clear navigation and easy interpretation

==============================
# DASHBOARDS AND ANALYSIS PERFORMED

Advanced Visualization Showcase
File: Adv VisualizationPOWERBI.pbix
Focus on non-standard visuals, drill-through, and visual hierarchy.

E-commerce Performance Dashboard
File: s4_ecommerce.pbix
Analysis includes revenue, orders, AOV, regional performance, and CLV.

Food Delivery Sales and Modeling
File: FoodDelivery_Sales_Charts_Modeling_06.pbix
Covers sales trends, delivery efficiency, profitability, and scalable data modeling.
